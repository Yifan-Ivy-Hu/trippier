# Trippier Application

This is an online tour booking service built using node.js, express, mongoDB, moogoose

## Demo

[See demo here](https://trippiers.herokuapp.com)

![pic1](/demo/pic1.png)

## Features

- sign up
- sign in
- log out
- update password
- forget and reset password
- online payment
- create reviews
- CRUD operations
