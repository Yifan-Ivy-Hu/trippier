# Trippier Application

This is an online tour booking service built using node.js, express, mongoDB, moogoose

## Demo

[See demo here]{trippiers.herokuapp.com}

## Features

- sign up
- sign in
- log out
- update password
- forget and reset password
- online payment
- create reviews
- CRUD operations
